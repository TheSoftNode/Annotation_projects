Test 1: Float overflow produces infinity
------------------------------------------------------------
Large float: 1.7e+308
Large * 10: inf
Is infinity: True
Is finite: False

Test 2: Infinity arithmetic (silent continuation)
------------------------------------------------------------
inf + 1000: inf
inf * 2: inf
inf - inf: nan (NaN)
System continues running with inf/NaN: True

Test 3: Integer overflow comparison
------------------------------------------------------------
sys.maxsize: 9223372036854775807
Integer overflow behavior is platform-dependent:

  - C int64_t: Undefined behavior or wrapping
  - Result: More explicit failure than float's silent inf
